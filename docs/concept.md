# 仕様書: security_game — 情報処理安全確保支援士レベルの知識が身につく学習ゲーム

## 概要

情報処理安全確保支援士(SC)試験レベルの高度情報処理技術者試験を想定した問題を組み込み、遊びながら勉強してセキュリティ知識を身につける学習ゲーム。

**方向性決定: 2026-07-08 代表回答** — 学習主体(勉強して知識を身につける形)。難易度基準は支援士レベル。

**方向性更新: 2026-07-20 代表回答** — ゲーム形式を**謎解きゲーム型のマップ攻略**に決定:

- 謎解きゲームのような形で、正答することでマップを進んでいく(**全10〜20マップ想定**)
- 各マップでは**攻撃側のハッカーの手段を探り、ヒントを探して防衛する**(攻撃を読み解く探索→防衛の流れ)
- **使う暗号をパズルのように組み合わせる**メカニクスを核にする

→ 従来案の「クイックドリル(午前II型)主体のMVP」から、謎解き×防衛シナリオ主体へ転換。ドリル・間隔反復は補助要素(復習モード等)として位置づけを再検討する。

## 背景・目的

- 座学だけでは定着しにくいセキュリティ知識を、ゲームの反復・達成感で継続学習に変える
- 代表は 2026年11月にネットワークスペシャリスト(NW)試験を受験予定。セキュリティ分野は NW でも配点が大きく、本ゲームでの学習がそのまま試験対策になる([[project_nw_exam_2026]]、study_agent と連携)
- 将来的には支援士試験そのものへの挑戦や、六創堂の教育・研修コンテンツへの転用も視野

## 学習コンテンツ

### 問題ソース

- IPA の支援士試験 過去問(午前II・午後)をベースにする。IPA 過去問は出典明記のうえ利用可能
- 分野: 暗号・認証・PKI、ネットワークセキュリティ(FW/IDS/IPS/WAF)、攻撃手法(XSS/SQLi/CSRF/標的型)、セキュアプログラミング、インシデント対応、法制度
- 将来: オリジナル問題・シナリオ問題を追加

### 出題形式

1. **クイックドリル**(午前II 型): 4択を1問ずつテンポよく。スキマ時間用
2. **シナリオステージ**(午後型): インシデント事例を読み進めながら設問に答えるアドベンチャー形式。午後問題の長文読解をステージ化する
3. **復習モード**: 間違えた問題を間隔反復(spaced repetition)で再出題。定着を担保する学習の本体

## ゲーム要素

- 正解で XP 獲得、レベルアップ。連続正解・連続学習日数(ストリーク)ボーナス
- 分野別の習熟度レーダーチャートで弱点を可視化
- ランクを試験の合格ライン(正答率60%)と対応させ、「合格圏」到達を目標化
- シナリオステージのクリアでストーリーが進む(セキュリティ担当者として事件を解決していく)

## MVP(最小構成)の提案

1. Web アプリ(スマホブラウザ対応)
2. 午前II 過去問のクイックドリル + 分野別成績 + 間隔反復復習
3. ストリーク・XP などの基本ゲーミフィケーション

→ まず代表の NW 試験対策(セキュリティ分野)として実用開始し、使いながらシナリオステージを育てる。

## 非機能要件(初期整理)

- スマホでスキマ時間に1問から遊べる軽さ
- 学習履歴の永続化(端末ローカル → 将来はアカウント同期)
- IPA 過去問の出典表記を問題ごとに明示

## スコープ外

- 実際の攻撃ツール提供・実環境への攻撃演習(擬似環境・設問内で完結)
- 対人対戦(将来候補)

## 決定事項(2026-07-08 代表回答)

- **プラットフォーム: Web アプリ**(スマホブラウザ対応)。Android アプリ化は必要になったら後から
